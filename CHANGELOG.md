## [1.0.0] - 2026-02-08

### Added
- Initial release
- Dynamic status display based on electricity price and solar production
- Beautiful gradient backgrounds for each state (Perfect, Good, Caution, Bad)
- Display current electricity price
- Show cheapest upcoming price in next 6 hours
- Click card to view detailed pricing information
- HACS compatibility
- Full documentation and examples

### Features
- Real-time status updates
- Customizable entities
- Responsive design
- Icon-based visual feedback
